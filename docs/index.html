Convert .epub file to .cbz file in browser

This web application is usually used to convert comic/manga. Because CBZ usually loads faster than ePub.

这个web应用可以将epub文件转换到cbz文件。一般cbz读取更快（但会丢失元数据）。

✏️ You can select multiple files. You can use 'save as...' to use this page offline.

可以选择多个文件。可以通过“另存为...”离线使用本页面。

⚠️ Please use latest Chrome/Edge/Firefox on PC. Other browsers like Saferi may have bugs.

请使用最新的Chrome/Edge/Firefox浏览器的PC版，其他浏览器可能没办法正常运行。
